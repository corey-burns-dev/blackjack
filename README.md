# Blackjack

A browser blackjack game built with Astro, React, and Tailwind CSS. It focuses on fast rounds, clean table UI, and casino-style game flow.

## Highlights

- Hit, stand, and surrender flow
- Natural blackjack payout handling
- Bankroll tracking and recharge flow
- Animated dealing and win effects

## Stack

- Astro
- React
- Tailwind CSS

## Quick start

```bash
bun install
bun run dev
```

Checks and build:

```bash
bun run check
bun run build
```
